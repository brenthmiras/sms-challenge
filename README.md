# react-starter

## Requirements
1. Node.js version 8.5.0
2. Npm     version 5.7.1

## Running the application
1. Navigate to project directory
```sh
$ cd react-starter
```
2. Install dependencies
```sh
$ npm install
```
3. Start the server
```sh
$ npm run start:dev
```
