# sms-challenge

## Requirements
1. Node.js version 8.5.0
2. Npm     version 5.7.1

## Running the application
1. Navigate to project directory
```sh
$ cd sms-challenge
```
2. Install dependencies
```sh
$ npm install
```

3. Build
```sh
$ npm run build
```

4. Start the server
```sh
$ node server.js
```

5. Browse to http://localhost:3000
